Section 5 - Forms and Tables

  • td = cell
  • tr = table row
  • th = table header
  • colSpan = for header with multiple colms
  • rowSpan = for header with multiple rows
  • Forms
    • Form action attribute specifies where data will be sent when the form is submitted
    • Form method attribute specifies which HTTP method should be used
    • Most common element is "input", Type attribute controls how input behaves, no closing tag
    • placeholder adds temp input in form
    • label element associates text with form
    • only one element per ID, using the "for" attribute, it links via ID
    • Button element, if it is nested inside of the Form element, then it will automatically try to submit the form if specified with the action attribute
    • Name attribute, like a variable for whatever goes in the input box
    • Checkbox and radio buttons, choose through type
    • Select element, drop-down menu, use Option inside Select to have a list in dropdown
